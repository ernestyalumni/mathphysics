# AGENTS.md — Srednicki Cadabra2 Sub-Agent Guide

Read this before touching anything in this directory.

## What This Is

Chapter-by-chapter Cadabra2 symbolic computation + LaTeX PDF notes for
Srednicki's *Quantum Field Theory*, building toward spinor-helicity and MHV
amplitudes. Each chapter produces a computation script and a LaTeX/PDF export.

## Chapter Pattern

Every chapter follows the same two-file pattern:

```
chNN_<topic>.py          # computation: Cadabra2 symbolic algebra + numpy
chNN_export_latex.py     # pulls results from above, generates .tex, compiles .pdf
```

Generated files (do NOT commit):
```
chNN_<topic>.tex         # generated by export script
chNN_<topic>.pdf         # compiled by pdflatex
chNN_<topic>.aux         # LaTeX build artifact
chNN_<topic>.out         # LaTeX build artifact
chNN_<topic>.toc         # LaTeX build artifact
chNN_<topic>.log         # LaTeX build artifact
```

**Only commit `.py` files and any `.md` files you create. Never commit build artifacts.**

## Running Scripts

All Cadabra2 scripts run inside Docker:

```bash
docker run --rm -it \
  -v /home/propdev/.openclaw/workspace/repos/Monoclaw:/Monoclaw \
  cadabra2-ubuntu:24.04 \
  python3 /Monoclaw/Python/Cadabra2/Srednicki/chNN_<topic>.py
```

For LaTeX export + PDF compile, same container:

```bash
docker run --rm -it \
  -v /home/propdev/.openclaw/workspace/repos/Monoclaw:/Monoclaw \
  cadabra2-ubuntu:24.04 \
  python3 /Monoclaw/Python/Cadabra2/Srednicki/chNN_export_latex.py
```

To compile `.tex` → `.pdf` manually (run twice for ToC):

```bash
cd Python/Cadabra2/Srednicki
pdflatex chNN_<topic>.tex && pdflatex chNN_<topic>.tex
```

## Key Conventions

- **Metric**: (+,−,−,−) throughout
- **ε convention**: ε^{12} = ε_{21} = +1
- **Cadabra2 LaTeX export**: use `Ex._latex_()` for expressions; `mat2pmatrix()` for numpy → pmatrix
- **σ matrices**: Srednicki's convention — σ^μ = (1, σ^i), σ̄^μ = (1, -σ^i)
- Srednicki uses g_{μν} = diag(−1,+1,+1,+1) in most places — check per chapter

## After Completing a Chapter

1. Run the computation script — verify it exits cleanly with no errors
2. Run the export script — verify `.pdf` compiles
3. Update `PROGRESS.md` — mark chapter done, note any caveats
4. Commit only `.py` files: `git add chNN_*.py && git commit -m "feat(srednicki): ch NN <topic>"`
5. Push branch — Ernest merges to master
