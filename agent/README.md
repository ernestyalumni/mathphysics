# MathPhysics Specialized Agent

This directory contains configuration for a **specialized OpenClaw agent** focused exclusively on mathematical physics research, specifically scattering amplitudes, spinor-helicity formalism, MHV amplitudes, and BCFW recursion.

### Purpose
- Provide a cleaner, more focused context when working on this repository.
- Separate physics research persona from general Monoclaw/sandbox tasks.
- Make it obvious to other OpenClaw instances what this agent is for.

### How to use
Other OpenClaw instances should read:
- `agent/SOUL.md` — personality and working style
- `agent/TOOLS.md` — project-specific tools and commands
- `TASKS.md` (root) — current tasks

This structure is intentional so any agent scanning the workspace can immediately understand the purpose of the `agent/` subdirectory.
